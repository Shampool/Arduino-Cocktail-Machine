# Cocktail Machine

## Components

1x Arduino UNO/Nano

1x I2C LCD1602

1x Rotary Encoder KY-040 

1x HX711 Load cell

2x L298N Motor Driver

3x Peristaltic Pump

1x DC Jack

1x 15V 5A Power Supply
